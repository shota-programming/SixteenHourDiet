# SixteenHourDiet - 16時間断食サポートアプリ

## 📱 アプリ概要

SixteenHourDietは、16時間断食（間欠的断食）をサポートするiOSアプリです。健康な生活習慣の確立をサポートし、体重管理と断食記録を一元化して管理できます。

## ✨ 主要機能

### 🕐 16時間断食タイマー
- **開始/停止機能**: ワンタップで断食タイマーを開始・停止
- **リアルタイム表示**: 残り時間を円グラフで視覚的に表示
- **完了通知**: 16時間達成時に成功通知を送信
- **進行状況**: 現在時刻と終了予定時刻を表示
- **既存記録チェック**: 同日の重複記録を防止

### ⚖️ 体重記録・管理
- **日次記録**: 毎日の体重を簡単に記録・更新
- **グラフ表示**: 週・月単位での体重変化を美しいグラフで可視化
- **履歴管理**: 過去の記録をカレンダーで確認
- **上書き機能**: 既存記録の更新が可能
- **5日ごと目盛り**: 月表示で見やすい5日間隔の目盛り

### 📊 履歴・統計
- **カレンダー表示**: 断食・体重記録をカレンダーで確認
- **詳細表示**: 日付を選択して詳細情報を表示
- **編集機能**: 過去の記録を編集可能
- **統計グラフ**: 体重変化の傾向を分析
- **ステータス表示**: 断食中・成功・失敗の正確な表示

### ⚙️ 設定・カスタマイズ
- **通知設定**: 断食成功通知のON/OFF
- **絵文字設定**: カレンダーに表示する絵文字をカスタマイズ
- **データ管理**: 記録数の確認と全データ削除
- **断食時間設定**: 断食時間の調整（12-24時間）

### 📱 広告・マネタイズ
- **バナー広告**: 画面下部に広告を表示
- **インタースティシャル広告**: 体重記録完了時に表示
- **広告削除**: ¥500で広告を完全に削除
- **収益化**: ユーザーフレンドリーな広告戦略

## 📁 プロジェクト構造

```
SixteenhourDiet/
├── Models/
│   ├── WeightRecord.swift      # 体重記録モデル
│   ├── DietRecord.swift        # 断食記録モデル
│   └── NotificationSettings.swift # 通知設定モデル
├── ViewModels/
│   └── WeightViewModel.swift   # 体重管理ビューモデル（週・月表示対応）
├── Views/
│   ├── TabView.swift           # メインタブビュー
│   ├── TimerView.swift         # 断食タイマー画面
│   ├── WeightView.swift        # 体重記録画面（週・月切り替え対応）
│   ├── HistoryView.swift       # 履歴画面（ステータス表示修正）
│   ├── SettingsView.swift      # 設定画面
│   ├── WeightChartView.swift   # 体重グラフ（5日ごと目盛り対応）
│   ├── DayDetailEditView.swift # 日付詳細編集
│   └── EmojiPickerView.swift   # 絵文字選択
└── Managers/
    ├── UserDefaultsManager.swift # データ管理
    └── NotificationManager.swift # 通知管理
```

## 🚀 インストール・実行

### 前提条件
- Xcode 15.0+
- iOS 17.0+
- Apple Developer アカウント（実機テスト用）

### セットアップ
1. リポジトリをクローン
```bash
git clone https://github.com/shota-programming/SixteenHourDiet.git
```

2. Xcodeでプロジェクトを開く
```bash
cd SixteenhourDiet
open SixteenhourDiet.xcodeproj
```

3. ビルド・実行
- シミュレーター: ⌘+R
- 実機: デバイスを選択して⌘+R
